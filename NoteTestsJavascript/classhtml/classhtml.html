Tipos de Declarações

  • var: declaração de escopo global-geral não específica(não muito utilizado)
  • let: declaração de escopo para variáveis locais.
  • const: declaração de variáveis constates.

Tipos de variável

  • Number: Variável que recebe NÚMEROS: let age = 25;
  • String: Variável que recebe TEXTOS: let name = "Maria";
  • Boolean: Variáveis que recebe VERDADEIRO ou FALSO: let Ativar = true;
  • Undefined: Variável declarada, porém não inicializada/não recebeu valor nenhum, ou seja, nenhum valor atribuído: var Nome;
  • Null: Variável que indica que está vazia: let currentUser = null; console.log(currentUser); // Output: null;
  • Object: Variável que armazena coleções de dados e entidades mais complexas: let person = {name: "João", age: 30};
  • NaN: Significa Not-a-Number, é uma espécie de valor usado para representar um resultado que não é um número válido. 2/"zero" = NaN
  • Infinity: valores ocorrem em operações aritméticas que resultam em números maiores do que o máximo representável em JavaScript. Ex.: 1 / 0


Variável global e local

  • Variável Global: variável acessível em qualquer parte do código, sendo qualquer variável declarada fora de uma função ou bloco é global. Risco: colisão de nomes
  • Variável Local: variável acessível apenas dentro do escobo que foi declarada como bloco ou função. Fornece mais segurança são aqueles declarados com o let ou const.

Entrada e Saida de Dados

Caixas de Entrada e Saída

  • prompt(''): Caixa de Entrada que solicita ao usuário para inserir algo.
  • confirm(''): Caixa de Entrada boleana que aceita somente 2 resultados: OK ou CANCELAR.
  • document.white(''): Caixa de Saída usado para escrever textos diretamente no documento HTML.
  • console.log(''): Caixa de Saída utilizado para imprimir mensagens informativas no console.
  • alert(''): Caixa de Saída usada para fornecer informações ao usuário ou para exibir avisos.

Operadores

Operadores Básicos

  • +: Soma.
  • -: Subtração.
  • *: Multiplicação.
  • /: Divisão.
  • (): Operador básico de agrupamento que junta expressões matemáticas, como (2 * 3) + 5 = 16.
  • %: Operador de resto que apresenta o resto de uma divisão. Ex.: let resultado = 10 % 3; //Resultado = 1.
  • ++ ou += 1: Operador que incrementa as parcelas dos valores.
  • -- ou -= 1: Operador que diminui o total por parcela.
  • **: Eleva o número. Ex.: var calculo = 2**3; //output: 8

Operadores de Comparação:

  • ==: verifica se os valores são só iguais. Ex.: a == b;.
  • ===: verifica se os valores são idênticos (iguais e do mesmo tipo de variável.). Ex.: a === b.
  • !=: verifica se valores não são iguais. Ex.: a != b.
  • !==: verifica se os valores não são iguais e não são do mesmo tipo de dado. Ex.: a !== b.
  • <: verifica se o valor é menor. Ex.: a < b.
  • >: verifica se o valor é maior. Ex.: a > b.
  • <=: verifica se os valores são menor ou igual. Ex.: a <= b.
  • >=: verifica se os valores são maior ou igual. Ex.: a >= b.

Operadores Lógicos

Obs.: a = 2 e b = 7; nome = "paulo"
  • &&: Operador AND que retorna verdadeiro se ambos as condições forem verdadeiras. Ex.: if(a < b && b > a){//resultado verdadeiro}.
  • ||: Operador OR que retorna verdadeiro se uma das condições forem verdadeiras. Ex.: if(a > b || b != a){//resultado verdadeiro, pois a condição 2 é true}.
  • !: Operador NOT de negação que inverte resultados. Ex.: if(!(nome == "paulo")){//não aparecerá as declarações esse bloco apesar de ser verdadeiro}else{//irá aparecer esse bloco}.

Tratamento de Dados em JavaScript
  • Number.parseInt(variávelX): Irá converter os valores inteiros em números inteiros; var variavelX = Number.parseInt(variavelY);
  • Number.parseFloat(variávelX): Irá converter os valores em um número de casa decimal; var variavelX = Number.parseFloat(variavelY);
  • Number(variávelX): Modo mais simplificado de conversão de números. var número = Number(n);
  • String(variávelX) ou n.toString(): Irá converter os número em String. Ex.: 2 em "2";
  • variavelX.length: Irá contar a quantidade de caracteres de uma String.
  • variavelX.toUpperCase || variavelX.toLowerCase(): Transforma os textos de maneira maiúscula e minúscula, respectivamente.
