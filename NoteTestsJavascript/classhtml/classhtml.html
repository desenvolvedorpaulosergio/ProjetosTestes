Tipos de Declarações

  • var: declaração de escopo global-geral não específica(não muito utilizado)
  • let: declaração de escopo para variáveis locais.
  • const: declaração de variáveis constates.

Tipos de variável

  • Number: Variável que recebe NÚMEROS: let age = 25;
  • String: Variável que recebe TEXTOS: let name = "Maria";
  • Boolean: Variáveis que recebe VERDADEIRO ou FALSO: let Ativar = true;
  • Undefined: Variável declarada, porém não inicializada/não recebeu valor nenhum, ou seja, nenhum valor atribuído: var Nome;
  • Null: Variável que indica que está vazia: let currentUser = null; console.log(currentUser); // Output: null;
  • Object: Variável que armazena coleções de dados e entidades mais complexas: let person = {name: "João", age: 30};
  • NaN: Significa Not-a-Number, é uma espécie de valor usado para representar um resultado que não é um número válido. 2/"zero" = NaN
  • Infinity: valores ocorrem em operações aritméticas que resultam em números maiores do que o máximo representável em JavaScript. Ex.: 1 / 0


Variável global e local

  • Variável Global: variável acessível em qualquer parte do código, sendo qualquer variável declarada fora de uma função ou bloco é global. Risco: colisão de nomes
  • Variável Local: variável acessível apenas dentro do escobo que foi declarada como bloco ou função. Fornece mais segurança são aqueles declarados com o let ou const.

Variável Simples e Composta:

  • Variável Simples: variável que armazena 1 valor por vez. Ex.: let nome = "Paulo"
  • Variável Composta: variável que armazena vários valores de uma vez. Ex.: let person = {name: "João", age: 30};

Entrada e Saida de Dados

Caixas de Entrada e Saída

  • prompt(''): Caixa de Entrada que solicita ao usuário para inserir algo.
  • confirm(''): Caixa de Entrada boleana que aceita somente 2 resultados: OK ou CANCELAR.
  • document.white(''): Caixa de Saída usado para escrever textos diretamente no documento HTML.
  • console.log(''): Caixa de Saída utilizado para imprimir mensagens informativas no console.
  • alert(''): Caixa de Saída usada para fornecer informações ao usuário ou para exibir avisos.

Operadores

Operadores Básicos

  • +: Soma.
  • -: Subtração.
  • *: Multiplicação.
  • /: Divisão.
  • (): Operador básico de agrupamento que junta expressões matemáticas, como (2 * 3) + 5 = 16.
  • %: Operador de resto que apresenta o resto de uma divisão. Ex.: let resultado = 10 % 3; //Resultado = 1.
  • ++ ou += 1: Operador que incrementa as parcelas dos valores.
  • -- ou -= 1: Operador que diminui o total por parcela.
  • **: Eleva o número. Ex.: var calculo = 2**3; //output: 8

Operadores de Comparação:

  • ==: verifica se os valores são só iguais. Ex.: a == b;.
  • ===: verifica se os valores são idênticos (iguais e do mesmo tipo de variável.). Ex.: a === b.
  • !=: verifica se valores não são iguais. Ex.: a != b.
  • !==: verifica se os valores não são iguais e não são do mesmo tipo de dado. Ex.: a !== b.
  • <: verifica se o valor é menor. Ex.: a < b.
  • >: verifica se o valor é maior. Ex.: a > b.
  • <=: verifica se os valores são menor ou igual. Ex.: a <= b.
  • >=: verifica se os valores são maior ou igual. Ex.: a >= b.

Operadores Lógicos

Obs.: a = 2 e b = 7; nome = "paulo"
  • &&: Operador AND que retorna verdadeiro se ambos as condições forem verdadeiras. Ex.: if(a < b && b > a){//resultado verdadeiro}.
  • ||: Operador OR que retorna verdadeiro se uma das condições forem verdadeiras. Ex.: if(a > b || b != a){//resultado verdadeiro, pois a condição 2 é true}.
  • !: Operador NOT de negação que inverte resultados. Ex.: if(!(nome == "paulo")){//não aparecerá as declarações esse bloco apesar de ser verdadeiro}else{//irá aparecer esse bloco}.

Operador Ternário:

  • Operador Ternário é uma forma compacta de expressar uma condição if...else em uma única linha. Estruturado por:
    condição ? valor_se_verdadeiro : valor_se_falso. Exemplo:
    let status = (idade >= 18) ? "Maior de idade" : "Menor de idade"; //output: se idade for igual a 15, o resultado seria "Menor de Idade".

Tratamento de Dados em JavaScript
  • Number.parseInt(variávelX): Irá converter os valores inteiros em números inteiros; var variavelX = Number.parseInt(variavelY);

  • Number.parseFloat(variávelX): Irá converter os valores em um número de casa decimal; var variavelX = Number.parseFloat(variavelY);

  • Number(variávelX): Modo mais simplificado de conversão de números. var número = Number(n);

  • String(variávelX) ou n.toString(): Irá converter os número em String. Ex.: 2 em "2";

  • variavelX.length: Irá contar a quantidade de caracteres de uma String.

  • variavelX.toUpperCase || variavelX.toLowerCase(): Transforma os textos de maneira maiúscula e minúscula, respectivamente.

  • variavelX.toFixed(n_casas decimais): apresentará um número com suas casas decimais. Ex.:

  • variavelX.replace('substituir', 'por'): substituirá os valores de acordo com os parâmetros da função.

  • text.trim();: Remove espaços em branco do início e do fim de uma string.

  • variávelArrayX.sort((a, b) => a - b) ou variavelX.sort()Organiza os dados de um Array na ordem crescente.

  • variávelArrayX.sort((a, b) => b - a)Organiza os dados de um Array na ordem decrescente.

  • variávelArrayX.push(valor)Irá atribuir um valor no Array no final. Ex.: nomes.push("Ana")

  • variávelArrayX.reverse(): Irá inverter a ordem das posições dos elementos do Array (não irá organizar de forma crescente ou decrescente)

  • variávelArrayX.pop() : Irá remover o ultimo elemento do Array

  • Math.max(...variávelArrayX) e Math.min(...variávelArrayX): Apresentará o maior e o menor valor de um Array, respectivamente.

  • variávelArrayX.reduce((a, b) => a+b)/variavelX.length: Apresentará a média do Array. Obs.: a e b são só parâmetros

  • variávelArrayX.indexOf(valorY) : Irá retornar os números das chaves do Array onde se encontra. Ex.: var idades = [2,4,6,8] alert(idades.indexOf(8))//Se tiver a chave do 7 irá retornar o valor dele, caso contrário, irá retornar "-1"
    Para o JavaScript, o "–1" significa que ele pesquisou dentro do vetor e não encontrou nenhuma ocorrência.

Métodos de Captura e Inserção de dados DOM

Métodos de Captura de Dados DOM

Observação: "window" representa a janela do seu navegador e o ".value" é usada para obter ou definir o valor de um elemento de formulário
  • getElementById('id de uma tag'): seleciona um elemento-tag por um ID:
    var name = window.document.getElementById('IdTagInput').value;

  • getElementsByClassName(className): Seleciona todos os elementos com uma determinada classe:
    var name = window.document.getElementsByClassName('ClassTag').value;

  • getElementsByTagName(tagName)[x]: Seleciona todos os elementos com uma determinada tag.
    var name = window.document.getElementsByTagName('p')[0]; //Nesse caso irá selecionar a primeira tag P que tiver na HTML

  • querySelectorAll(selector): Seleciona todos os elementos que correspondem a um seletor CSS.
    var name = window.document.querySelector('.teste'); //Observe que o "." captura os dados de todas as tags que possuir uma classe chamada "teste"; Id = #.

Inserção de Dados DOM:

  • variavelX.innerHTML: insere valores nas tags não desprezando as configurações das tags:
    variavelX.innerHTML = "Nome: Paulo"; //Na execução, o nome "Paulo" ficará em sublinhado;

  • variavelX.innerText: insere valores nas tags, porém despreza as configurações das tags:
    • variavelX.innerText = "Meu nome é Paulo"; //O nome não ficará em negrito
  • variavelX.textContent: insere valores nas tags de para textos, porém despreza as configurações das tags
    • variavelX.textContent = "Lorem [...]"; //Texto grande!!!

Evendos Clássicos DOM:

  • mouseenter: disparará um evento quando o mouse passar por cima de algum elemento;
  • mousemove: disparará um evento quando você mover o mouse dentro do elemento, sendo disparado várias vezes;
  • mousedown: disparará um evento quando você clicar e segurar o mouse no elemento;
  • mouseup: disparará um evento quando você soltar o mouse do elemento já clicado;
  • click: disparará um evento quando você simplesmente clicar e soltar rapidamente;
  • mouseout: disparará um evento quando você tirar o mouse no elemento;
Obs.: para você inserir um desses eventos, basta adicionar o prefixo "on-" no início desses eventos e depois, vai em JavaScript. Segue a seguinte estrutura:

Laços de Repetições

Tipos de Laços de repetições:

  • for: Estrutura de Repetição com Variável de Controle: Estrutura =>
    for(inicialização; condição; incremento){//repetição}
    for(var count = 1; count <= 10; count++){//execução em 10 vezes}
  • while: Estrutura de Repetição com um Teste Lógico no Início
    while(condicao){//repeticao}
  • do..while: Estrutura de Repetição com um Teste Lógico no Final
    do{repetição}while(condição)
  • for(declaração in variávelArrayX){bloco-código}: tipo de laço de repetição para elementos Objects e Variáveis Compostas, especificamente.
    for(var count in nomes){console.log(nomes)}//será apresentado todos os nomes da variável "nomes". Significa que para cada contagem de chaves que estiver na variável nomes, será apresentado os valores da variável nomes.

Criação de Elementos HTML em JavaScript

Criando Elemento:

  • variavelX = document.createElement('nomeTag'): irá criar um elemento:
    • let opcao = document.createElement('option');

Atribuindo Valor:

  • variavelX.text = "": permite atribuir um valor a um elemento criado no JS
    opcao.text = "olá"

Atribuindo Elemento-Valor em uma tag HTML:

  • variavelY.appendChild(variavelX): permite atribuir o elemento em outra tag.
    elementoSelect.appendChild(opcao) //percebe-se que foi atribuido uma OPTION no SELECT, cujo valor é "olá".

Atribuindo ID/Class no ElementoJS

  • opcao.id = "nomeID": irá atribuir um ID no elementoJS
  • opcao.ClassName = "nomeCLASS": irá atribuir uma CLASS no elementoJS
  • Como observação, caso queira inserir várias classes em um único ElementoJS, você deve inserir
    opcao.classList.add("Classe1", "Classe2")..
